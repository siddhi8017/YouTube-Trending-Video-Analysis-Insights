# YouTube Trending Video Analysis & Insights

## 📌 Project Overview
This project analyzes trending YouTube videos to identify patterns in views, likes, and audience engagement.

## 📊 Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization of trends
- Insights generation

## 📈 Key Insights
- Videos with higher views tend to have more likes
- Engagement increases with popularity
- Certain categories dominate trending lists

## 🛠 Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
